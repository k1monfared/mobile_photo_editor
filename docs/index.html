Real-time Photo Adjustments

All adjustments are processed on the GPU using OpenGL ES 2.0 shaders ported from Telegram's photo editor. Changes appear instantly as you drag the slider.

  • Exposure, Contrast, Saturation, Warmth
  • Highlights, Shadows for tonal control
  • Sharpen, Fade, Grain, Vignette for creative effects
  • Each parameter has a centered or one-sided slider matching its natural range

Crop and Rotate

Precise crop control with a Telegram-style overlay. Drag corners or edges to resize. Pinch to zoom, pan to reposition. The rotation wheel gives you fine control from -90 to +90 degrees.

  • 6 aspect ratio presets: Free, 1:1, 4:3, 3:4, 16:9, 9:16
  • 90-degree rotation with icon button (tap 4x to return to start)
  • Horizontal mirror toggle
  • Expand mode: allow black areas when rotating, or auto-zoom to fill
  • Reset button restores all transforms to original

Freehand Drawing

Draw directly on your photo with a stamp-based brush engine. Strokes accumulate smoothly with configurable opacity for a natural painting feel.

  • 8 colors with visible selection indicator (white border)
  • Adjustable brush size (2px to 100px)
  • Opacity slider for transparent brush strokes
  • Eraser tool with icon button and toggle highlight
  • Undo up to 20 strokes (button dims when stack is empty)

Save to Gallery

Your edited photo is saved at full resolution to your device gallery. The filter pipeline renders the final image on the GPU at the original bitmap dimensions, then writes it as a high-quality JPEG.

  • JPEG quality 95 for minimal compression artifacts
  • Saved via Android MediaStore (visible in gallery apps)
  • Full resolution output (no downscaling)
  • All adjustments baked into the final image

Why ImgEdt?

GPU-Powered

All image processing happens on the GPU via OpenGL ES 2.0 GLSL shaders. Smooth 60fps preview even on mid-range devices.

Open Source

GPL v2 licensed. Full source code on GitHub. Algorithms ported from Telegram's battle-tested photo editor.

Lightweight

Under 7 MB APK. No ads, no tracking, no internet permissions required. Just a clean photo editing tool.

Library-Ready

Two-module architecture: the editor is a separate Android library (AAR) that can be embedded in other apps.

Install

Download the latest APK from GitHub Releases and install it on your Android device (5.0+).

For automatic updates, use Obtainium and point it at this repository.

Build from Source

git clone https://github.com/k1monfared/imgedt.git
cd imgedt
./gradlew assembleDebug
# APK at app/build/outputs/apk/debug/app-debug.apk
